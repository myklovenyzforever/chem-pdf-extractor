## Repository Description

Extract structured experimental data from scientific PDFs into Excel/CSV.

## About Section

Open-source tool for extracting structured literature data from scientific PDF papers, focused on chemical engineering, catalysis, materials, and environmental research. Supports configurable fields, Excel/CSV export, cache reuse, Ollama, and OpenAI-compatible APIs.
