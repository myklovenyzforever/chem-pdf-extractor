# v0.1.0 - Initial open-source release

This is the initial open-source release of Chem-PDF-Extractor.

## Highlights

- Batch PDF-to-Markdown/text conversion workflow.
- Configurable LLM-based extraction fields.
- Excel/CSV export for structured literature data.
- Bad-row filtering, suspicious-data records, and error logs.
- Resumable processing with Markdown and extraction cache reuse.
- Local configuration template without any built-in real API key.
- Synthetic examples and lightweight tests.
- Windows one-click workflow preserved for bundled release packages.

## Safety Notes

This release does not include real API keys.

Example data is synthetic and does not represent real published papers.

Bundled Windows runtime packages, if provided, should be attached as release assets and should not be committed to source control.
