# Examples

## English

The example files are synthetic. They are only used to demonstrate field templates and output formats.

They do not contain real copyrighted papers or real extracted research results.

- `sample_fields.json` provides chemistry/catalysis-oriented extraction fields.
- `sample_output.csv` shows the expected CSV output structure.
- `sample_output.xlsx` shows the expected Excel output structure.

Use these files to understand the shape of inputs and outputs, not as scientific data.

# 示例说明

## 中文

本目录中的示例文件都是合成数据，只用于展示字段模板和输出格式。

这些文件不包含真实版权论文，也不代表真实论文提取结果。

- `sample_fields.json`：化工/催化方向字段模板。
- `sample_output.csv`：CSV 输出结构示例。
- `sample_output.xlsx`：Excel 输出结构示例。

这些文件只能作为格式参考，不能作为科研数据使用。
