# Materials Synthesis Benchmark

Case ID: `materials_synthesis`

This is a synthetic/public-safe benchmark case. It does not contain real paper
text, a real DOI, private data, local paths, or API keys.

The case describes a fictional oxide composite synthesis workflow and provides a
small golden output table for first-pass extraction checks.

Files:

- `input.md`: synthetic source text.
- `fields.json`: extraction fields for this case.
- `golden_output.csv`: expected first-pass row.
