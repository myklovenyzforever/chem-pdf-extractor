# Code of Conduct

## Our pledge

We aim to keep Chem-PDF-Extractor a respectful and constructive open-source project for researchers, students, developers, and users working with scientific literature data.

## Expected behavior

- Be respectful and constructive.
- Discuss technical issues clearly.
- Help keep examples reproducible and privacy-safe.
- Do not pressure others to share private PDFs, unpublished manuscripts, API keys, logs, or confidential data.
- Respect copyright and data privacy.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Sharing other people's private data without permission.
- Posting API keys, tokens, private PDFs, unpublished manuscripts, or confidential outputs.
- Repeatedly derailing technical discussions.

## Reporting

If you see inappropriate behavior or unsafe sharing of sensitive data, please open a GitHub issue with a sanitized description, or contact the maintainer through the repository profile if public reporting would expose sensitive information.
