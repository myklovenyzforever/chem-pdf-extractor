# Release Package Notes

The GitHub repository does not commit `YiLaiHuanJing/` because it is a large local runtime folder.

For non-programming Windows users, a release package may include:

- `ShuJuTiQuJiaoBen.py`
- `YiJianQiDong.bat`
- `YiLaiHuanJing/`
- `README.md` or a short user guide

Do not include:

- real API keys
- `config.local.json`
- user PDF files
- output Excel files
- logs
- caches

Before publishing a release package, test it on a clean Windows machine or a clean folder.
